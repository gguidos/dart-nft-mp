# MICRO-SERVICE Gateway

The Gateway is responsible for handle security and redirecting requests to the respective endpoint(EP).

## Technologies

* NodeJS: Version 18.x.x

* Redis: Version 7.0.5

* MongoDB: Version 6.0.1

## Instalation

1. git clone the project
2. npm install
3. npm start
